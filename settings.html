Attachment Remover

Automatically removes user-sent image/file attachments from the chat once the AI has sent its response. This helps keep the chat log clean.

THIS IS IMPORTANT: If attachments are NOT disappearing, you MUST adjust the CSS selectors at the top of the index.js file (in the CONFIG.selectors object) to match your specific SillyTavern version's HTML structure. Use your browser's F12 developer tools to "Inspect" the chat elements (e.g., messages and attachment previews) to find the correct class names or IDs.
